FAQ

Preguntas frecuentes

¿Incluye dominio y hosting?

Sí. Todos los planes incluyen dominio y hosting por 1 año. Luego solo se paga la renovación anual del servicio.

¿Cuánto tarda el desarrollo?

Normalmente entre 7 y 21 días, dependiendo del plan elegido y de qué tan rápido se envíen los textos, fotos o información del negocio.

¿Cómo es la forma de pago?

Se comienza con una seña inicial para iniciar el desarrollo. El saldo restante se paga cuando la web está terminada y lista para publicarse.

¿Puedo pedir cambios en la web?

Sí. Durante el desarrollo vas a poder revisar la web y pedir cambios antes de la entrega final.

Cada plan incluye revisiones:

  • Plan básico: hasta 3 revisiones
  • Plan profesional: hasta 5 revisiones
  • Tienda online: hasta 10 revisiones

Una revisión significa que ves la web y enviás una lista de cambios. Todos los cambios se realizan juntos.

¿Después de publicada la web puedo hacer cambios?

Sí. Podés contratar un plan de mantenimiento mensual para actualizar tu sitio, hacer cambios o agregar contenido.

¿La web va a aparecer en Google?

Sí. La web se entrega optimizada para que Google pueda indexarla. También podés contratar el servicio de posicionamiento inicial.
